# python38_proj_adata
 Scrapy scraper
 
 It uses python 3.8 
 Install requirements with:
 ```
 pip install -r requirements.txt
 ```
It saves urls with additional information for all deputats.
To run the scraper type:
```
cd src
```
```
cd parliamentbg
```
```
scrapy crawl parliament_all -O quotes.json
```
